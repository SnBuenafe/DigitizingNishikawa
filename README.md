# DigitizingNishikawa
Repository for Digitizing Nishikawa et al (1985) Data

This repository digitizes spawning ground data on the following species:
1. Yellowfin tuna
2. Swordfish
3. Albacore
4. Skipjack tuna
5. Blujefin tuna
6. Southern bluefin tuna
7. Bigeye tuna
8. Blue marlin
9. Black marlin
10. Striped marlin
11. Sailfish
12. Shortbill spearfish
13. Frigate tuna
14. Little tuna
15. Bonitos
16. Slender tuna
17. Longfin escolar
18. Sauries
